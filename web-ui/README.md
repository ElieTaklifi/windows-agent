# Web UI Dashboard

This folder contains a standalone security-oriented dashboard for viewing inventory JSON output.

## Features
- Dashboard cards for total/high/medium/low execution surfaces.
- Security-centric risk estimation for quick triage.
- Filter by type, source, scope, and risk level.
- Search across all serialized entry fields.
- Table view for detailed review.

## Usage
1. Open `web-ui/index.html` in a browser.
2. Load an exported `inventory.json` file using the file picker.
3. Use filters and search to investigate potential high-risk surfaces.
